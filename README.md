> - 서브 도메인 : 연결이 비공개로 설정되어 있지 않습니다 : nginx에 서버를 입력할 때 기존 server 안에 코드를 작성했다. 항상 각각의 서버는 서로를 포함하는 것이 아니라 구분하여 작성해야 함
